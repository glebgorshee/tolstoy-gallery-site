Этот файл — самодостаточная память по проекту. Если открыл репозиторий в новом аккаунте/на новом компе и ничего не помнишь — читай его, тут всё.

## Что это
Статический сайт-визитка галереи современного искусства «Толстой» (Москва, Новинский бул., 1/2).
Двуязычный (RU/EN). Референс визуала (выбрал заказчик) — **edenart.com**.

- **Прод:** https://glebgorshee.github.io/tolstoy-gallery-site/
- **Репозиторий:** github.com/glebgorshee/tolstoy-gallery-site (публичный, аккаунт `glebgorshee`)
- **Локально:** `~/Desktop/tolstoy-gallery-site/`
- **Хостинг:** GitHub Pages (есть `.nojekyll`; ассеты грузятся с `?v=<время сборки>` для сброса кеша).
  Pages пересобирается до ~15 мин после push; HTML кешируется у клиента ~10 мин (жёсткий refresh покажет свежее).

## Стек и визуал
- Чистый HTML/CSS/JS, без сборщиков. GSAP + Lenis (плавный скролл, параллакс героя).
- Шрифты: **Noto Serif** (заголовки) + **Manrope** (текст) — Google Fonts.
- Белый фон, чёрный текст, много воздуха. Название галереи по центру шапки = ссылка на главную.
- Кнопки-капсулы. Liquid-glass — только на кнопке поверх медиа (герой).

## ГЛАВНОЕ: сайт собирается генератором, НЕ руками
`build.py` строит все `.html` из данных. **Никогда не правь `.html` напрямую** — они перезапишутся.

Порядок работы:

(в Claude Code есть сервер `tolstoy` в `~/.claude/launch.json`, порт 5501).

## Двуязычность RU/EN
Каждый переводимый текст = атрибуты `data-ru` / `data-en`. JS `applyLang()` подменяет textContent (для input — placeholder), выбор хранится в localStorage `agt-lang`. Переключатель EN/RU в шапке.
Список художников сортируется по отображаемому имени; при смене языка JS пересортировывает по `data-sru`/`data-sen`.

## Работы
- **Главный источник новых работ — `data/artworks.json`** (07.2026): `{ "<key>": [ {slug, title, tech_ru, tech_en, size, sold, imgs:[имена файлов]} ] }`. Файлы лежат в `assets/img/works/<key>/`. Несколько imgs = карусель ракурсов. Доливается ко всем художникам поверх catalog.json/julie_works.json.
- catalog.json — работы старого сайта (accardi/tamburro/van-apple), julie_works.json — конфеты Julie, kiko — авто-глоб всей папки.
- Художник без работ → плейсхолдер «Работы скоро», в фильтрах коллекции его нет.
- **Видео-работы**: mp4 в `assets/img/works/<key>-video/` → отдельный блок «Видео-работы» на странице художника (bashev — 14, van-apple — 2 X-Ray). Названия — из имени файла или словарь `VIDEO_TITLES` в build.py.
- Размеры: единицы автоматом RU «см» / EN «cm» (см. work_tile). Техники старого каталога переводятся словарём `TECH_RU`.

## Производительность (07.2026)
- `build.py` при сборке генерит **миниатюры** `assets/img/thumbs/**` (800px, magick q76, progressive; фолбэк sips) и **постеры видео** `assets/img/thumbs/posters/**` (ffmpeg, 480px). Пересоздаются только устаревшие (mtime).
- Плитки/карточки грузят thumbs, лайтбокс — оригинал (`data-images`/`data-full`), карусель в плитке листает thumbs (`data-thumbs`).
- Видео-плитки: preload="none" + poster; на тач-устройствах автоплей по видимости (IntersectionObserver), на десктопе hover.
- В лайтбоксе из коллекции/главной — ссылка «Имя художника →» (`.lc-artist`), на странице самого художника скрыта.

## Вкладка «Видео» (15.07.2026)
Отдельная страница `video.html` (в навигации после «Коллекции»), собирает ВСЕ видео сайта в один блок — «бегущие колонки», как на glebgorshe.
- **Эффект:** 3 колонки, встречная авто-прокрутка на чистом CSS (`@keyframes vidwall-scroll`, `translateY(calc(-50% - 6px))`, дубль контента для бесшовного цикла). Скорость задаётся инлайн `animation-duration` ∝ числу видео в колонке. Направления `up/down/up`.
- **Механика (main.js, IIFE `.vidwall`):** видео `preload="none"` + poster; play/pause по IntersectionObserver (с guard `innerHeight>0` от headless) + пинок по первому жесту; клик → видео-лайтбокс `#vlb` со звуком (стена всегда muted).
- **Агрегация (`build_video()`):** галерейные `assets/video-gallery/*.mp4` + все `assets/img/works/*-video/*.mp4` (Башев 14, Ван Эппл 4). Источники чередуются, чтоб колонки были разнообразными. Постеры для галерейных видео генерит `make_derivatives()` (добавлен glob на `video-gallery`).
- **Хостинг:** пожатые mp4 в репозитории (8 промо ≈ 23 МБ). Если видео станет много/тяжело — переносить на Cloudinary `dgletcy0i` (CDN), в `src` подставить cloudinary-URL, механика не меняется.
- Ван Эппл видео-папка теперь 4: x-ray-suitcase, x-ray-birkin, x-ray-backpack, x-ray-birkin-red.
