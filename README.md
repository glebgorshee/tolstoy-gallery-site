# Art Gallery Tolstoy — сайт (рыба)

Статический сайт галереи. Референс визуала — [edenart.com](https://www.edenart.com/):
Noto Serif (заголовки) + Manrope (текст), белый фон, чёрный текст, много воздуха,
фулл-блид герои, мозаичная сетка работ, лайтбокс.

## Стек
Чистый HTML/CSS/JS + GSAP (reveal) + Lenis (плавный скролл). Без сборки, без зависимостей.
Хостится как статика (Vercel/Netlify), потом прикрутить домен.

## Структура
- `index.html` — главная (герой, интро, художники, избранные работы, CTA визита)
- `artists.html` — список художников
- `artist-<slug>.html` — 6 страниц художников (генерятся)
- `collections.html` — вся коллекция с фильтром по художнику + лайтбокс
- `visit.html` — контакты, режим по записи, форма-заглушка, карта
- `assets/css/style.css`, `assets/js/main.js`
- `assets/img/` — работы (`works/<artist>/`), портреты, оформление
- `data/catalog.json` — каталог работ (title / техника / размер / SOLD)
- `build.py` — генератор: правишь тексты/художников там и пересобираешь

## Пересобрать
```
python3 build.py
```

## Локальный просмотр
```
python3 -m http.server 5501
# http://localhost:5501
```
